# Services Layer

This directory contains application services and integrations.

## Structure

- **auth/** - Authentication service (login, logout, token management)
- **api/** - API client and HTTP service
- **storage/** - Local storage service (Hive, Secure Storage)
- **notification/** - Notification service (Push, Email, SMS)
- **websocket/** - WebSocket service for real-time features
- **payment/** - Payment gateway integration
- **ai/** - AI service integration (recommendations, matching, analytics)
